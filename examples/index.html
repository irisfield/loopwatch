loopwatch

event-loop measurement tools · build first, then serve from project root

Measure Loop Lag

measureLoopLag

Wraps a function and measures event-loop lag, RAF cadence, and long tasks while it runs. Spikes in lag mean the main thread was blocked.

Select a scenario and click Run.

Long Tasks

LongTaskObserver

Observes tasks that hold the main thread ≥ 50 ms. Trigger blocks during observation to see them collected.

Click Observe, then trigger blocks during the window.

LoopMonitor

continuous

Runs lag, RAF cadence, and long-task collection on a repeating cycle. Reports continuously until stopped.

Click Start to begin continuous monitoring.

Before / After

compareReports

Measures loop lag before and after a blocking operation, then diffs the reports. Negative deltas = improvement.

Select a scenario and click Run.
